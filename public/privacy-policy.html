TradeView 隱私權政策

Privacy Policy

最後更新日期：2024 年 12 月 | Last Updated: December 2024

1. 概述 | Overview

TradeView（以下簡稱「本應用」）是一款投資組合管理工具，協助您追蹤和管理投資資產。本隱私權政策說明我們如何處理您的資料。

TradeView (the "App") is an investment portfolio management tool that helps you track and manage your investment assets. This Privacy Policy explains how we handle your data.

2. 資料收集 | Data Collection

重要聲明 | Important Notice

本應用採用離線優先架構（Offline-First Architecture），不會收集、儲存、上傳或傳輸任何個人資料或交易數據到任何伺服器。

This App adopts an offline-first architecture and does NOT collect, store, upload, or transmit any personal information or transaction data to any server.

2.1 我們不收集的資料 | Data We Do NOT Collect

  • 個人身份資訊（姓名、電子郵件、電話號碼等）
  • 交易記錄和投資數據
  • 財務資訊（帳戶餘額、資產配置等）
  • 位置資訊
  • 裝置識別碼
  • 任何其他個人可識別資訊
  • Personal identification information (name, email, phone number, etc.)
  • Transaction records and investment data
  • Financial information (account balances, asset allocation, etc.)
  • Location information
  • Device identifiers
  • Any other personally identifiable information

2.2 資料儲存位置 | Data Storage Location

所有應用資料（包括交易記錄、資產配置、帳戶資訊等）完全儲存在您的裝置本地，使用瀏覽器的本地儲存（Local Storage）功能。這些資料：

  • 僅存在於您的裝置上
  • 不會上傳到任何雲端服務或伺服器
  • 不會與第三方共享
  • 僅由您本人控制和管理

All app data (including transaction records, asset allocation, account information, etc.) is completely stored locally on your device using the browser's local storage functionality. This data:

  • Only exists on your device
  • Is not uploaded to any cloud service or server
  • Is not shared with third parties
  • Is controlled and managed solely by you

3. 網路使用 | Network Usage

本應用需要網路連線權限，僅用於以下目的：

  • 獲取即時股價資訊：從公開的金融數據來源獲取股票價格，用於計算您的投資組合價值
  • 獲取匯率資訊：獲取美元與台幣的即時匯率
  • 載入應用程式：當使用網頁版或 TWA（Trusted Web Activity）版本時載入應用程式資源

This App requires network connectivity permissions only for the following purposes:

  • Fetch real-time stock prices: Retrieve stock prices from public financial data sources to calculate your portfolio value
  • Fetch exchange rates: Retrieve real-time USD/TWD exchange rates
  • Load the application: Load application resources when using the web version or TWA (Trusted Web Activity) version

注意 | Note

這些網路請求僅用於獲取公開的金融市場數據，不涉及傳輸任何個人資料或交易記錄。

These network requests are only used to fetch public financial market data and do not involve transmitting any personal information or transaction records.

4. 資料備份 | Data Backup

本應用提供資料匯出功能，讓您可以將所有資料匯出為 JSON 檔案並儲存在您的裝置或雲端儲存服務中。這個備份過程：

  • 完全由您手動執行
  • 備份檔案完全由您控制
  • 備份檔案不會自動上傳到任何伺服器
  • 您可以選擇將備份檔案儲存在任何位置

This App provides data export functionality that allows you to export all data as a JSON file and store it on your device or cloud storage service. This backup process:

  • Is completely performed manually by you
  • Backup files are completely under your control
  • Backup files are not automatically uploaded to any server
  • You can choose to store backup files anywhere

建議 | Recommendation

為了避免裝置損壞或瀏覽器資料被清除導致資料遺失，我們強烈建議您定期使用應用內建的「備份資料」功能進行備份。

To prevent data loss due to device damage or browser data clearing, we strongly recommend that you regularly use the built-in "Backup Data" feature to create backups.

5. 第三方服務 | Third-Party Services

本應用可能會從第三方服務獲取公開的金融市場數據（如股價、匯率）。這些第三方服務可能會有自己的隱私權政策。本應用僅讀取這些公開數據，不與第三方服務共享您的個人資料或交易記錄。

This App may retrieve public financial market data (such as stock prices and exchange rates) from third-party services. These third-party services may have their own privacy policies. This App only reads these public data and does not share your personal information or transaction records with third-party services.

6. 資料安全 | Data Security

由於所有資料都儲存在您的本地裝置上，資料安全主要取決於您的裝置安全措施。我們建議您：

  • 使用裝置的螢幕鎖定功能（如密碼、指紋、臉部識別）
  • 定期備份資料
  • 在共用裝置上使用應用後清除瀏覽器資料
  • 妥善保管備份檔案

Since all data is stored locally on your device, data security mainly depends on your device security measures. We recommend that you:

  • Use your device's screen lock features (such as passwords, fingerprints, face recognition)
  • Back up data regularly
  • Clear browser data after using the app on shared devices
  • Keep backup files safe

7. 兒童隱私 | Children's Privacy

本應用不主動收集任何個人資料，因此也不存在收集兒童個人資料的情況。由於所有資料都儲存在本地，家長應監護兒童使用本應用。

This App does not actively collect any personal information, so there is no situation where children's personal information is collected. Since all data is stored locally, parents should supervise children's use of this App.

8. 政策更新 | Policy Updates

我們可能會不定期更新本隱私權政策。重大變更時，我們會在應用中或透過其他適當方式通知您。建議您定期查看本政策以了解最新的隱私權保護措施。

We may update this Privacy Policy from time to time. For significant changes, we will notify you in the app or through other appropriate means. We recommend that you review this policy regularly to stay informed about the latest privacy protection measures.
